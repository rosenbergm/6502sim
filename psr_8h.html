◆ PSR_INITIAL_VALUE

constexpr unsigned char PSR_INITIAL_VALUE = 0b00100100
constexpr

The initial value of the processor status register (P).

On the real chip, no exact boot state is guaranteed. We set the (I) bit high because we do not support hardware interrupts.
